# validation

バージョン: v1.0.0（過去PJ）
作成日: 2026-01-22
既存: src/lib/validation.ts

## Functions(関数)

### validatText
- テキストの必須チェック
- パラメータ: text: string
- 未入力時: '入力してください。'
- 正常時: ''

### validatSelect
- セレクトボックスの必須チェック
- パラメータ: select: string, defaultSelect: string
- 未選択時（select === defaultSelect）: '選択してください。'
- 正常時: ''

### validatRadio
- ラジオボタンの必須チェック
- パラメータ: select: string
- 未選択時（select === ''）: '選択してください。'
- 正常時: ''

### validatCheckbox
- チェックボックスの必須チェック
- パラメータ: checkbox: string[]
- 未選択時（checkbox.length === 0）: '選択してください。'
- 正常時: ''

### validatRfcEmail
- メールアドレスのRFC準拠チェック
- パラメータ: email: string
- チェック項目:
  - 未入力: '入力してください。'
  - @未含: '@が含まれていない場合は無効なメールアドレスです。'
  - @連続: '@が2つ以上連続しています。'
  - @複数: '@が2つ以上含まれています。'
  - ローカル部65文字以上: '@より前の部分が、65文字以上になっています。'
  - ドメイン254文字以上: '@より後の部分が254文字以上の長さになっています。'
  - 全体255文字以上: 'メールアドレス全体で255文字以上になっています。'
  - ローカル部.連続: '.(ドット)が、@より前の部分で、2つ以上連続しています。'
  - ローカル部.先頭/末尾: '.(ドット)を、ローカル部分の最初と最後(@の直前)に使っています。'
  - -先頭/末尾: '-(ハイフン)が先頭・末尾または、@の前後にあります。'
  - ドメイン部-と.隣接: '@より後の部分で、-(ハイフン)と.(ドット)が隣接しています。'
  - ドメイン数字のみ: '@より後の部分が、数字だけで構成されています。'
  - 特殊文字含む: '特殊文字(( ) , : ; < > [ ])が含まれています。'
  - ドメイン.なし: '.(ドット)が、@より後の部分で1つもありません。'
- 正常時: ''

### validatKana
- 全角カタカナチェック
- パラメータ: kana: string
- 未入力時: '入力してください。'
- 非カタカナ時: '全角カタカナで入力してください。'
- 正規表現: /^[ァ-ンー　]+$/
- 正常時: ''

### validatNumber
- 半角数字チェック
- パラメータ: number: string
- 未入力時: '入力してください。'
- 非数字時: '半角数字で入力してください。'
- 正規表現: /^[0-9]+$/
- 正常時: ''

## Dependencies(依存関係)

- なし（Pure JavaScript）
