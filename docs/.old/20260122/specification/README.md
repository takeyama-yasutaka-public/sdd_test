# 過去PJ仕様書

バージョン: v1.0.0
作成日: 2026-01-22

## Overview(概要)

このディレクトリは過去プロジェクトの既存コードからAIが生成した仕様書です。
SDDドキュメントに準じた形式で記述しています。

## Structure(構成)

/components
- ui-kit: デザインコンポーネント（Button, Heading, Pager等）
- ui-features: ロジック統合コンポーネント（Pager, Header等）

/lib
- ユーティリティ関数仕様書

/app
- ページコンポーネント概要
