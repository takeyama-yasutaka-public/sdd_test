# 仕様駆動開発(SDD) ドキュメント

このディレクトリは仕様駆動開発(Specification-Driven Development)の設計ドキュメント群です。

## ディレクトリ構成

/ui
- UIデザインシステム関連
- コンポーネント、ページ、デザイントークンの仕様書

/features
- ビジネスロジック関連
- API、ユーティリティ関数の仕様書

/env.example
- 環境変数設定例

/schemas
- データスキーマ定義

/plan
- プロジェクトプランドキュメント

/.example
- SDDテンプレート・サンプル
