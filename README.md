# WEB CORPORATION

Next.js 15.x + React 19 + Tailwind CSS で構築されたコーポレートサイト

## 技術スタック

- **フレームワーク**: Next.js 15.5.9
- **UIライブラリ**: React 19.2.3
- **スタイリング**: Tailwind CSS 4.1.18
- **言語**: TypeScript 5.x
- **状態管理**: Zustand 5.0.8
- **テスト**: Jest 30.2.0, React Testing Library 16.3.0

## 開発方法

このプロジェクトは仕様駆動開発（SDD）に基づいて開発されています。

- 仕様書: `/docs` ディレクトリ
- ワークフロー: `/docs/.example/sdd/workflow.md`
- 仕様書記述ルール: `/docs/.example/sdd/specification-example.md`

## セットアップ

```bash
npm install
npm run dev
```

## テスト

```bash
npm test
```
